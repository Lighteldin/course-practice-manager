## About The Project

An application that allows user to add questions for each course's chapter and the option to solve them all at once. Created in 2022 when I first started learning Java as a beginner.
The project is not intended to be used but rather to practice Java & SQL with.
<p align="right">(<a href="#readme-top">back to top</a>)</p>



### Built With

* Java
* Swing/AWT
* H2 SQL Database

<p align="right">(<a href="#readme-top">back to top</a>)</p>
